Simple commands

Let's quickly go through some simple commands. You probably won't use them alot, however they are still there and you should know what they do.

  • /ping

    Displays ping from host system to Discord API. Does not display command latency.

  • /aboutme

    Displays some general information about the bot. (I guess it's useless if you had read through the main page of this website)

  • /changelog

    Displays recent changes in the bot.

  • /help

    This is probably the command which had lead you to this website.

  • /userinfo

    Displays some general information about a user.

And that's it! But we're not done yet. Now it's time for some more interesting features.

Interface with Rule34

This is probably the biggest feature of the bot. It contains some relatively complicated commands.

You should also check the Rule34 cheatsheet if you don't know how its search works.

Main commands

  • /r34 search

    This is the command you probably will use the most from this command group

    Allows you to search for content from Rule34. Returns an interface which will allow you to look through the posts you searched for.

    It fully supports Rule34 search, so if you want to filter the search results better, you'll need to check that cheatsheet.

    Using this command will cost you 2 tokens! Using the interface is free though.

    Command arguments:
    1. Tags - A list of tags to search posts by. Supports full Rule34 search.
    2. Page - A page to start from. Depends on size. Default value - 0.
    3. Size - size of the page. (1-10) Default value - 2.
  • /r34 tagsearch

    This is the second command you'll use the most from this group.

    Allows you to search for tags by inputing a key. Usefull if you don't really know the actual tag you need.

Other commands

Let's quickly go through other commands in this group, which you'll use less frequently than the ones mentioned before.

  • /r34 randomtag

    This command will return a random Rule34 tag from a specified tag category

  • /r34 defaulttags command group

    Commands from this group allow you to edit tags, which will be automaticly added to all of your searches.

    Usefull when you want to filter unwanted content.

  • /r34 supply

    Allows you to setup automatic supply of content from Rule34.

    You need to have administrator permissions to use this command.

Interfaces

  • Search interface

    When /r34 search is used, it returns a message with a simple UI you can use to go through posts from Rule34.

    It contains 4 buttons:

    1. Quit - delete the message with the interface.
    2. Previous - go to the previous page.
    3. Next - go to the next page.
    4. Browse - returns a URL to rule34 website which will contain posts from the current page.

  • Tagsearch interface

    Has a button, which allows to search for more tags by the provided key and returns the result if any tags were found.

    The button destroys itself when used.

Economy & tokens

Introduction

The bo has its own economy system since it was updated to v0.5b The currency is called "Tokens". You can claim 100 tokens daily. Tokens can be transfered between registered users.

In future, there will be more ways to gain and spend tokens, but for now you can be a part of this relatively simple system.

Bank

To view your ballance, claim your daily tokens and check your taxes, you can use the bank interface. It can be called with /bank command.

The interface has a few buttons. I'm sure you are not that dumb to figure them out by yourself.

Taxes

Yep, you have to pay taxes. To be more accurate, the tax will be collect automaticly at 00:00 UTC.

Total tax equals to 25% of your ballance. It will be collect only if it reaches a value of 100 tokens. (This means you need to have at least 400 tokens) So make sure to spend your tokens, or they simply will go to waste.

Commands

  • /bank

    Displays the bank interface.

  • /transfer

    Allows you transfer tokens to another user. Can be used 3 times per day.
