Quick introduction

Tina3 provides a lot of features. It may be hard to understand some of them. I recommend you to go through this documentation to understand how to use the bot at its full potential.

Simple commands

Let's quickly go through some simple commands. You probably won't use them alot, however they are still there and you should know what they do.

  • /ping

    Displays ping from host system to Discord API. Does not display command latency.

  • /aboutme

    Displays some general information about the bot. (I guess it's useless if you had read through the main page of this website)

  • /changelog

    Displays recent changes in the bot.

  • /help

    This is probably the command which had lead you to this website.

  • /userinfo

    Displays some general information about a user.

And that's it! But we're not done yet. Now it's time for some more interesting features.

Interface with Rule34

This is probably the biggest feature of the bot. It contains some relatively complicated commands.

You should also check the Rule34 cheatsheet if you don't know how its search works.

Main commands

  • /r34 search

    This is the command you probably will use the most from this command group

    Allows you to search for content from Rule34. Returns an interface which will allow you to look through the posts you searched for.

    It fully supports Rule34 search, so if you want to filter the search results better, you'll need to check that cheatsheet.

    Using this command will cost you 2 tokens! Using the interface is free though.

    Command arguments:
    1. Tags - A list of tags to search posts by. Supports full Rule34 search.
    2. Page - A page to start from. Depends on size. Default value - 0.
    3. Size - size of the page. (1-10) Default value - 2.
